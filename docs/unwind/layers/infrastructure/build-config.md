# Build & Dependencies

## Package manifest

### package.json [MUST] <!-- id: file:package.json:package.json -->

[package.json](https://github.com/cliftonc/rascal/blob/master/package.json#L1-L69)

`name: rascal`, `version: 21.0.1`, `main: index.js`, `license: MIT`,
`author: Stephen Cresswell`. CommonJS library — no build/transpile step; source
ships as-is. Repo/homepage point at `onebeyond/rascal`.

**Runtime dependencies:**

| Dependency | Version | Purpose |
|------------|---------|---------|
| async | ^3.2.4 | Async control flow |
| debug | ^4.3.4 | Namespaced debug logging |
| forward-emitter | ^0.1.1 | Re-emit events across emitters |
| generic-pool | ^3.8.2 | Channel pooling |
| lodash | ^4.17.21 | Utilities |
| lru-cache | ^7.10.1 | Redelivery counter cache |
| safe-json-parse | ^4.0.0 | Safe content parsing |
| stashback | ^2.0.1 | RPC reply correlation |
| uuid | ^8.3.2 | Message/correlation ids |
| xregexp | ^5.1.0 | Binding-key regex parsing |

**Peer dependency:** `amqplib >=0.5.5` (the underlying AMQP driver — host app
must provide it).

**Dev dependencies:** amqplib ^0.10.2, chalk ^4.1.2, chance ^1.1.8,
eslint ^8.45.0, eslint-config-airbnb-base ^15.0.0, eslint-plugin-import ^2.27.5,
husky ^8.0.3, lint-staged ^11.2.4, nyc ^15.1.0, random-readable ^1.0.1,
zunit ^4.0.0.

**Engines:** `node >=14` (note: CHANGELOG 21.0.0 says minimum is now 16; the
`engines` field still reads `>=14`).

**Scripts:**

| Script | Command | Purpose |
|--------|---------|---------|
| test | `zUnit` | Run zUnit test suite |
| lint | `eslint .` | Lint |
| lint:fix | `eslint --fix .` | Lint + autofix |
| lint-staged | `lint-staged` | Lint staged files (used by pre-commit) |
| coverage | `nyc --report html --reporter lcov --reporter text-summary zUnit` | Coverage |
| docker | `docker run -d --name rascal-rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.12.9-management-alpine` | Start local RabbitMQ for dev/test |
| prepare | `husky install` | Install git hooks on `npm install` |

**Test runner config (`zUnit` key):** `pollute: true`, `pattern: ^[\w-]+.tests.js$`.
**lint-staged:** `**/*.js` → `eslint --fix`.

## Lint / format config

### .eslintrc.json [SHOULD] <!-- id: file:.eslintrc.json:.eslintrc.json -->

[.eslintrc.json](https://github.com/cliftonc/rascal/blob/master/.eslintrc.json#L1-L26)

Root ESLint config. Extends `eslint-config-airbnb-base`, `env.node`,
`parserOptions.ecmaVersion: 2018`. Disables ~13 airbnb rules (max-len,
consistent-return, no-plusplus, no-param-reassign, etc.). `no-shadow` allows
`cb`/`err`; `no-unused-vars` ignores `debug` var and `_`-prefixed args.

### test/.eslintrc.json [SHOULD] <!-- id: file:test/.eslintrc.json:.eslintrc.json -->

[test/.eslintrc.json](https://github.com/cliftonc/rascal/blob/master/test/.eslintrc.json#L1-L18)

ESLint overrides for the test directory. Declares zUnit globals as readonly
(`describe`, `xdescribe`, `odescribe`, `it`, `xit`, `oit`, `before`,
`beforeEach`, `after`, `afterEach`, `include`) and disables `no-shadow`.

### README.md [SHOULD] <!-- id: file:README.md:README.md -->

[README.md](https://github.com/cliftonc/rascal/blob/master/README.md#L1-L1924)

Primary documentation (~1900 lines). Describes rascal's value over raw amqplib
(config-driven vhosts/exchanges/queues/bindings, cluster support, content
parsing, encryption, reconnection, retry/redelivery, channel pooling, flow
control, RPC, promise + callback APIs) and documents the full configuration
surface. The canonical human reference for the public API and config — not a
rebuild contract itself but describes one.
